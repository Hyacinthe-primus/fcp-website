# File Converter Pro - Website

The official website for [File Converter Pro](https://file-converter-pro.org/), a free offline file converter for Windows.

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Font Awesome 6.4 (CDN)
- Google Fonts (DM Sans, DM Mono)
- GitHub Pages deployment

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page |
| `features.html` | Feature showcase with visuals |
| `download.html` | Download options and donation modal |
| `roadmap.html` | Development roadmap |
| `docs.html` | Full documentation |

## Local Development

No build step required. Open any `.html` file in a browser or run a local server:

```bash
python -m http.server 8000
```
